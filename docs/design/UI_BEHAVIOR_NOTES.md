# UI Behavior Notes

Home / Dashboard
- Hiển thị danh sách cycles (templates + user cycles)
- Nút tạo cycle mới rõ ràng; chọn template nhanh

Cycle Builder
- Thêm/sắp xếp phases bằng drag hoặc nút
- Mỗi phase có tên, thời lượng (số giây), type (focus/rest/breathing)

Timer Screen
- Hiển thị phase hiện tại, thời gian còn lại, progress ring
- Màu nền thay đổi theo loại phase
- Controls: Start / Pause / Resume / Skip / Stop

Settings
- Theme (Light/Dark), âm thanh bật/tắt (v1: tắt), export/import (future)
