# User Stories

1. Khách có thể dùng app ngay mà không cần đăng ký.
2. Người dùng tạo/sửa/xóa cycle với nhiều phase tùy chỉnh.
3. Người dùng chạy timer, Pause/Resume, Skip phase.
4. Người dùng đăng nhập bằng email để đồng bộ cycles lên Firestore.
5. Có 3 template mặc định để bắt đầu nhanh.
6. Nút Start bị vô hiệu nếu cycle không có phase.

Ghi chú: các stories này tương ứng với PRD bản tóm tắt, giữ ngắn để ưu tiên phát triển.
